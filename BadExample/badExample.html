Violations of Best Security Practices

  • Using a GET instead of a POST method on a form. When using a GET method all of the users data will be displayed in the url
  • Sensitive Data Exposure: Keeping sensitive data unencrypted
  • Exposing users sensitive data
  • Not having mandatory HTTPS

How to Prevent Security Risks

  • Encrypting users sensitive data(Passwords, credit card info)
  • Mandatory HTTPS for everything
