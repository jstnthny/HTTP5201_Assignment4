Code Linting

Code linters are utilized to help validate code. Code Linters differ from traditional validators because it catches mistakes and errors that are still "technically" allowed but are not exactly up to coding standards.

For this site I installed ESLint which is a tool for identifying and reporting patterns found in the ECMAScript/JavaScript code.

Unit Testing

Unit testing is one of the most common types of tests that decelopers write. The way unit testing works is by importing a file into a test file and using a testing framework or assertion library to write these tests and by using node to run them.

Examples of Unit Testing Frame Works:

  • Mocha
  • Jest
  • Jasmine
  • NUnit

Git & Version Control

Version control is the practice of tracking and managing changes to code. The most popular and commonly used distributed version control system is git.

This website was made using version control while every major change would be pushed to a git repository. The repository can be found here.
